# FILTRANDO DADOS COM WHERE E LIKE
```sql
SELECT NOME, TELEFONE FROM CLIENTE;
```

# FILTRANDO 
```sql
SELECT NOME, SEXO FROM CLIENTE
WHERE SEXO = 'M';

SELECT NOME, ENDERECO FROM CLIENTE
WHERE SEXO = 'F';

SELECT NOME, ENDERECO FROM CLIENTE
WHERE ENDERECO = 'RJ';
```

# UTILIZANDO O LIKE
```sql
SELECT NOME, SEXO FROM CLIENTE
WHERE ENDERECO LIKE 'RJ';
```

# CARACTER CORINGA % -> QUALQUER COISA
```sql
SELECT NOME, SEXO FROM CLIENTE
WHERE ENDERECO LIKE '%RJ';
```
